watch 是一个非常实用的命令，基本所有的 Linux 发行版都带有这个小工具，如同名字一样，watch 可以帮你监测一个命令的运行结果，省得你一遍遍的手动运行。

直接在 watch 后面接你想运行的命令，watch 就会帮你重复运行，并把每次的结果都更新在屏幕上。

默认 watch 会以 2s 的间隔重复运行命令，你也可以用 -n 参数指定时间间隔～

还有一个实用的参数是 -d，这样 watch 会帮你高亮显示变化的区域，这样更加一目了然了～
Ctrl+c 就可以退出～

你可以拿他来监测你想要的一切命令的结果变化



$echo -e "a\nb\nc\nd"
-e 选项用于输出转义字符，另外echo默认会在字符串输出结束的时候在末尾加换行
例如以上会输出
a
b
c
d
$

echo 的几个颜色
前景颜色 重置0、黑色30、红色31、绿色32、黄色33、蓝色34、洋红35、青色46、白色47
背景颜色 重置0、+10同上
举例：
$echo -e "\e[1;32m Hello World \e[0m"

开始 \e[1;32m 结束回复原来 \e[0m

cat -s file # 压缩文件中连续空白行
cat -n file # 显示行号

find . -print   以\n为结束符打印文件名
find . -print0  以\0为结束符
其中-print可忽略

-o或条件,-a与条件 注意括号与选项之间必须有空格
find . \( -name "*.txt" -o -name "*.php" \) -print

find 选项
-iname  按名称忽略大小写
-regex
-iregex
!           否定关键词
-type       按文件类型搜索 f 文件 d目录 l 链接文件

-atime      最近一次访问时间
-ctime      创建时间
-mtime      最后一次修改时间

-size       +/- 2k 文件大小

-delete     删除动作

-user       按用户
-perm       按权限

-exec       执行命令
{}          find结果

例如 find . -name "*.sh" -exec cat {} \; 其中\; 不可缺少
